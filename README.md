Project Overview

This project uses Machine Learning and NLP techniques to classify text documents into multiple categories. TF-IDF Vectorization and Logistic Regression are used for multi-label prediction.

Objectives

- Build a multi-label text classification model.
- Apply NLP techniques for text processing.
- Predict multiple categories for a document.
- Automate text organization.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Categories

- Technology
- Education
- Sports
- Health

Workflow

1. Load the dataset.
2. Preprocess text and labels.
3. Apply TF-IDF Vectorization.
4. Train the Logistic Regression model.
5. Evaluate performance.
6. Predict categories for new text.

Output

- Predicted Categories
- Accuracy Score
- Classification Report
- Category Distribution Chart
